# Ablauf

1. **Einstieg**  
   - Nutzer öffnet Webanwendung ohne Login  

2. **Erste Abfrage**  
   - Basisangaben werden erfasst (Alter, Geschlecht -> weiblich: stillend oder schwanger)  

3. **Symptomerfassung**  
   - Auswahl über Körpermodell (mehrere Körperregionen möglich)  
   - Auswahloptionen (Checkboxen)  
   - Option für psychische Beschwerden  
   - Freitext möglich, falls kein passendes Symptom vorhanden ist  

4. **Dynamische Nachfragen**  
   - System stellt weitere Fragen basierend auf den Symptomen (Dauer, Stärke, Begleitsymptome, Vorerkrankungen, Medikamenteneinnahme)  

5. **Erweiterte Angaben (optional)**  
   - Zusätzliche Informationen zur genaueren Einschätzung (Gewicht, Allergien)  
   - Abfrage ist überspringbar

6. **Auswertung / Triage**  
   - Bewertung der Situation  

7. **Ergebnis**  
   - Anzeige einer Handlungsempfehlung (Hausarzt, Notdienst, Notfall)
   - inklusive Begründung  

8. **Weiterleitung / Aktionen**  
   - Terminbuchung  
   - Rezeptanfrage  
   - Telefonweiterleitung  

9. **Daten & Abschluss**  
   - Daten werden gespeichert  
   - Nutzer erhält Code für späteren Zugriff
